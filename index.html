Introduction

Trump Speech Editor is a javascript project that allows you to mute sections of a youtube video.

The Editor

The editor allows you to select sections of a YouTube video and generate a URL to visualize the video with the selected sections silenced.

The Viewer

The viewer is responsible for playing the video and for controling the video's volume. It receives a video_id and a list of non overlaping, ordered, time intervals to mute.

Know Bugs

  • The editor is not checking if the id provided by the user is a valid Youtube video id.
  • The player is not handling when the user moves the video forward or rewind it.
  • If the list of intervals to mute is unordered or overlaped the viewer may not work.
  • There is probably more bugs, but the MVP is working fine.
